      AuthForm - методы

      Методы

      ButtonAuth_Click Логика авторизации при нажатии на кнопку авторизации.
      ButtonAuthLogic Логика авторизации пользователя.
      CheckboxShowAuthPassword_CheckedChanged Показывает или скрывает пароль в зависимости от состояния чекбокса.
      CreateAccountLabel_MouseHover Изменяет цвет текста при наведении мыши на метку создания аккаунта.
      CreateAccountLabel_MouseLeave Восстанавливает цвет текста при уходе мыши с метки создания аккаунта.
      Dispose Clean up any resources being used.
      (Переопределяет FormDispose(Boolean))
      InitializeComponent Required method for Designer support - do not modify the contents of this method with the code editor.
      LabelCloseApp_Click Закрывает приложение при нажатии на метку закрытия приложения.
      LabelCreateAccount_Click Открывает форму регистрации при нажатии на метку создания аккаунта.
      TextboxPassword_KeyPress Обрабатывает нажатие клавиши Enter в поле пароля для вызова логики авторизации.
